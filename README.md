# Year 9 Science Revision - Complete Site

Static GitHub Pages website containing Biology, Chemistry and Physics topic pages.

Upload the contents of this folder to a GitHub repository and enable GitHub Pages from the branch containing `index.html`.

## Structure

- `index.html` links to all topics.
- `biology/` contains four Biology topic pages.
- `chemistry/` contains four Chemistry topic pages.
- `physics/` contains four Physics topic pages.
- Subject-specific scripts, styles and exact organiser images are kept at the root.

## Leaderboard

Students enter their name before starting a quiz. Scores are stored in the browser using `localStorage`, so using the same name on the same device/browser adds new correct answers to that student's previous total.
